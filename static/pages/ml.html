Model Architecture

Stacked Ensemble: Layer 1 consists of a Random Forest (100 trees, depth 5) and a Gradient Boosting Machine (100 estimators, depth 3). Their probability outputs feed a Calibrated Logistic Regression meta-learner (CalibratedClassifierCV, 3-fold).


Validation: 4-fold Walk-Forward TimeSeriesSplit — no look-ahead bias. Each fold trains on earlier history and tests on later data, mimicking real trading conditions.


Target: Forward return > 0.5% over the chosen horizon. Probability output is calibrated so 55% = genuine edge, not just rank.


Data source: Uses Upstox daily OHLCV from the screener cache when available, otherwise falls back to Yahoo Finance (2 years).


Features (16): RSI(7), RSI(14), MACD histogram, Bollinger %B, ATR%, volume ratio (5d/20d), EMA distances (9/20/50), 1d/5d/20d returns, 52-week high/low distances, vol continuity (ATR5/ATR20), CLV.
